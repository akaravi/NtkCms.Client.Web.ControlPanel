اطلاعات پروژه

این اساسی ترین و فرم پیش فرض دارای بخش های فرم است. برای اضافه کردن بخش فرم از کلاس.form-section با هر تگ عنوان استفاده کنید. این فرم دکمه هایی در گوشه پایین سمت چپ دارد که موقعیت پیش فرض آن است.

پروفایل کاربر

شما همیشه می توانید رنگ مرز کنترل های فرم را با استفاده از border-* کلاس در این مثال ما کاربر داریم border-primary کلاس برای کنترل فرم. دکمه های عمل فرم در سمت راست پایین قرار دارند.

اطلاعات تماس و بیو

ردیابی موضوع

این فرم راهنمایی های مربوط به شناور را برای ارائه اطلاعات مفیدی در حالی که کاربر پر می کند، نشان می دهد. از ویژگیهای داده مانند تلنگر استفاده کنید data-toggle، ماشه data-trigger، تعیین سطح data-placement، عنوان data-title برای نشان دادن راهنمایی ها در کنترل های فرم.

جدول زمانی

این فرم نشان می دهد استفاده از آیکون ها با کنترل های فرم. موقعیت نماد با استفاده از را تعریف کنید has-icon-left یا کلاس has-icon-right. استفاده از کلاسicon-* برای تعریف آیکون برای کنترل فرم. بخش آیکن ها را برای لیست آیکون هایی که می توانید استفاده کنید مشاهده کنید.
